# geraciador de tarefas
API RESTful para gerenciamento de tarefas com autenticação JWT, testes automatizados e integração com MongoDB via Docker.

---

## Tecnologias
- [Node.js](ca://s?q=Node.js) + [Express](ca://s?q=Express.js)
- [MongoDB](ca://s?q=Como_funciona_o_MongoDB) (rodando em container)
- [JWT](ca://s?q=JSON_Web_Token) para autenticação
- [Jest + Supertest](ca://s?q=Como_funciona_Jest_e_Supertest) para testes
- [Docker Compose](ca://s?q=Explicar_Docker_Compose) para orquestração
